# Task 4 - Logistic Regression Classification

This repository contains a completed solution for **Task 4: Classification with Logistic Regression** (from the provided PDF task).  
It uses the Breast Cancer Wisconsin dataset (from scikit-learn) to build a binary classifier using logistic regression, evaluate it, and demonstrate threshold tuning.

Files:
- `app.py` — runnable script that trains the model, prints evaluation metrics, and saves a model.
- `requirements.txt` — Python package requirements.
- `report.md` — short report with results and explanations.
- `tools_used.md` — list of tools and libraries used.
